# Arena
?> Represents a clash royale arena.

#### Attributes
> **`name`** - The name of the arena.    
**Returns:** str

> **`number`** - The number of the arena. 4 for P.E.K.K.A. Playhouse, for example.    
**Returns:** str

> **`trophies`** - The number of trophies required to get to this arena.    
**Returns:** int
